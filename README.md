# SIEN Public Site

High-end public website for `sien.co.ke`.

## Environment

```txt
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
NEXT_PUBLIC_SITE_URL=https://www.sien.co.ke
```

If Supabase variables are missing, the site uses static SIEN project data bundled in `data/static-projects.ts`.

## Vercel

Root directory: `apps/site`
Build command: `pnpm build`
